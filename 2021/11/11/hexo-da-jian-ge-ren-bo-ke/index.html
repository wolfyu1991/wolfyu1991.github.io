Hexo安装配置

安装Hexo

$ npm install -g hexo

初始化

$ hexo init blog
$ cd blog

生成，本地测试

$ hexo g # 生成
$ hexo s # 启动服务

上传到Github

只上传生成的blog静态文件

$ hexo d -g #生成并上传

访问站点

HelloWorld！

Hexo常用命令

常用命令

hexo new "postName" #新建文章
hexo new page "pageName" #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）
hexo deploy #部署到GitHub
hexo help  # 查看帮助
hexo version  #查看Hexo的版本

命令缩写

hexo n == hexo new
hexo g == hexo generate
hexo s == hexo server
hexo d == hexo deploy

组合命令

hexo s -g #生成并本地预览
hexo d -g #生成并上传
